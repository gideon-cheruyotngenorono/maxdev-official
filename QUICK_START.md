## 🚀 Deploy to Vercel in 5 Minutes

### Option 1: Automatic (Recommended)

1. **Push to GitHub**
   ```bash
   cd "C:\Users\Administrator\MAX DEV"
   git init
   git add .
   git commit -m "MAX DEV Portfolio with 11 Projects"
   git remote add origin https://github.com/YOUR_USERNAME/max-dev-portfolio.git
   git push -u origin main
   ```

2. **Connect to Vercel**
   - Go to https://vercel.com/new
   - Import your GitHub repository
   - Click "Deploy"
   - Done! 🎉

### Option 2: Manual Deployment

1. Visit https://vercel.com
2. Click "New Project"
3. Upload your project folder
4. Configure (leave defaults)
5. Deploy

## 🔧 Customization Tips

### Add New Project
Edit `projects.html` section and add to `projectsData`:
```javascript

    emoji: "🎯",
    title: "PROJECT NAME",
    price: "KES XX,000",
    // ... other fields
}
```

### Change Colors
Modify Tailwind colors in `<head>` tailwind-config script

### Update Links
Change navigation links across all HTML files

### Modify Pricing
Update `priceValue` and `price` fields in projects data
